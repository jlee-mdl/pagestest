MDL Login Instructions
Document M001; Last Document Revision: May 21, 2015

Purpose

This document provides simple instructions on how to log into the NEW MDL system. If you have any questions or issues with system access, please email support@mydealerlot.com or call 888-635-7343 , Ext. # 1.

Important Note

Please ensure that any previous shortcuts, bookmarks, favorites, or anything of the sort which was previously set up for MyDealerLot access is removed from your computer.

How to Access MDL’s New Login Page

There are two simple ways to access:

METHOD 1: Direct Access

Open Internet Explorer and go to mysdc.mydealerlot.com
** Note: DO NOT TYPE WWW.



METHOD 2: From MDL’s Web Site Home Page




The below screen will be displayed after accessing the above web address.

To set up a desktop icon in Internet Explorer:

  • Right Click anywhere on the page outside of the live boxes (shown below)
  • Select Create Shortcut “Shown by Red Arrow”

Once you enter your email and password, click SIGN IN. If you have any access issues, please email support@mydealerlot.com with your full name and dealership name.
